Tidy Validator/Textpad
Installation Instructions

  

One of the most useful features of TextPad is the editor's ability to work with external tools. Several people have asked me how to set up TextPad to work with the HTML Tidy Web page checker. Here's how to do it:

  1. Download the Tidy HTML Validator Executable file and install it to the TextPad program folder.
  2. In TextPad, go to the Configure menu and choose Preferences.
  3. On the left side of the Preferences dialog window, click Tools.
  4. On the right side of the dialog, click Add and choose Program from the drop-down list that appears.
  5. A file dialog appears. Find your HTML Tidy executable (for example, c:\Program Files\TextPad\tidy.exe) and click Open.
    You're returned to the Preferences dialog.
  6. Click Apply on the Preferences dialog window. Tidy is now added to the list of tools.
  7. On the left side of the Preferences dialog window, click the + sign by Tools and then click Tidy.
  8. In the Parameters field of the Preferences dialog window, enter the Tidy command line parameters you use (for example, -muc) before $File (which appears in the Parameters field by default). Keep $File in the Parameters field because it tells HTML Tidy to check the document you're working on. For example:

  1. -mi -asxhtml --uppercase-tags no --uppercase-attributes no $File

  2. Select the Capture output option. This will display any messages from HTML Tidy in a new window.
  3. Click Apply and then click OK.

To use HTML Tidy, open an HTML file in TextPad and select Tidy from the Tools menu. A new window should open and list the messages from HTML Tidy. Close this window, and then click Yes on the dialog box that appears.
