Objective:

This assignment is a more advanced experience with event driven programing. Generally, event driven programs require the construction of a GUI (Graphical User Interface) first followed by the "glue" programming code to make the program operate. The purpose of this program will be to do chip counting for a casino, determine winnings, and calculate tax if neccessary based on the requirements described below.

Requirements:

A gambling casino in Tinian, CNMI needs a program that will help its cashiers do customer cash out transactions. Good programmers are hard to find in the CNMI, so they have approached you. JavaScript has been chosen for the programming environment, because it can simply be placed on a web server for access. They have agreed to fly you over for an all expense paid week to observe the casino in action.

You enjoy your time in Tinian while doing your Problem Solving Phase "research" and begin by outlining your "Program Specs". You have agreed that you will not leave Tinian until all parties agree with your Program Specifications Report. Once the report is approved you will enter the Implementation Phase and return home to do your coding. The casino has agreed to pay you one red chip per hour until you complete the program.

The program must satisfy the new US Federal IRS Requirement that if the gambler is a U.S. Resident, twenty percent of gambling winnings, must be withheld when cashing in chips at the cashier's window. However, CNMI residents will only have to pay five percent withholding tax and foreign residents will have no tax withheld. Additionally, the casino wants to have the chip counting automated.

Therefore, the program must meet the following specifications:

  1. All program code must be JavaScript embedded in a valid xhtml strict document containing a GUI form.
  2. Create a GUI using XHTML forms that will allow a user to input multiple receipts for chips purchased. All receipts should be displayed in a multiline textarea.
  3. Next in the form GUI the user will enter total of chips in the following denominations.
    1. Black = $5
    2. Blue = $20
    3. Red = $50
    4. Green = $100
    5. Gold = $500
  4. The total value of all colored chips will be calculated and displayed.
  5. The program will then subtract this amount from the total amount the gambler started with to determine the amount won.
  6. The gamblers residency status will be entered. If gambler is a U.S. Resident or CNMI resident the Social Security Number also needs to be entered.
    1. CNMI Residents will have 5% withholding tax applied to the winnings.
    2. Per IRS regulations, 20% will be withheld from the winnings if gambler is a US Resident (Including Guam).
    3. Non-residents will have no tax withheld from winnings.
  7. The program will display in the form the winnings, tax withheld, and cash received.
  8. This form will be printed as a receipt, which should be kept by the gambler for their tax records.
